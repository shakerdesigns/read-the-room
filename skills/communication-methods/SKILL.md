# Communication Methods

This skill provides reference material for choosing between validation (Imago Dialogue) and analytical (SCR) communication approaches.

## Core Decision

Before communicating, read the room. The person's state determines your approach.

- **C-States** (Calm, Curious, Clear...) → Analytical approach (SCR)
- **T-States** (Testy, Tense, Triggered...) → Validation approach (Imago) first

## Quick Reference

### SCR Framework (Analytical)

Use when they're receptive:
1. **Situation** — What's happening (agreed facts)
2. **Complication** — Why it matters (create tension)
3. **Resolution** — What you recommend

### Imago Dialogue (Validation)

Use when they're defensive:
1. **Mirror** — "What I'm hearing is..."
2. **Validate** — "That makes sense because..."
3. **Empathize** — "I imagine you feel..."

Then transition to analytical.
